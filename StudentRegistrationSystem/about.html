About Project

Built for Day 1 Flask practice

The project focuses on frontend layout, template rendering, navigation, and clean page design.

Project Overview

The Student Registration System is a Flask application that presents a home page, a student registration form, a records page with dummy data, and an about page. It is prepared for later database integration.

Features

  • Flask routes using render_template()
  • Responsive navigation across all pages
  • Professional registration form interface
  • Styled table with 10 dummy student records
  • Reusable color theme, cards, buttons, and footer

Team Information

Developed by the Student Registration System team as part of the Innolift Day 1 web development task using Flask, HTML, and CSS.
