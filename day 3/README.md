# Day 3 - NumPy, Pandas, CSV, and EDA

This folder contains the Day 3 practice programs:

- `numpy_marks_analyser.py` - analyzes student marks using NumPy.
- `dataframe_builder.py` - creates and explores a Pandas DataFrame.
- `csv_explorer.py` - loads and previews CSV data.
- `missing_data_detective.py` - detects and handles missing data.
- `group_compare.py` - groups records and compares summary values.
- `full_eda_report.py` - generates a complete exploratory data analysis report.

Supporting files:

- `utils_eda.py` - shared helper functions for EDA scripts.
- `disease_outbreak_sample.csv` - sample CSV dataset used by the examples.
- `.txt` files - text copies of the same code examples.
